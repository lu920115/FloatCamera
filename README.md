# FloatCamera

A minimal floating camera app for macOS.

FloatCamera lets you keep a small camera window on top of your screen, so you can record tutorials, presentations, or talking-head videos more easily.

## Features

- Always-on-top floating camera window
- Simple and lightweight
- Designed for quick recording use cases
- Minimal interface

## System Requirements

- macOS 14.0 or later

## Installation

1. Download `floatcamera.dmg`
2. Open the DMG file
3. Drag **FloatCamera** into **Applications**
4. Open the app from Applications

If macOS shows a security warning the first time:
- Right click the app
- Choose **Open**
- Then confirm again

## Camera Permission

FloatCamera needs camera access to display your live camera preview.

If permission has not been granted:
- Open **System Settings**
- Go to **Privacy & Security**
- Open **Camera**
- Enable permission for **FloatCamera**

## Notes

This is a minimal release shared for simple use.
It may not include future updates or long-term maintenance.

# FloatCamera

一个适用于 macOS 的极简悬浮相机工具。

FloatCamera 可以让你的摄像头画面以小窗形式悬浮在屏幕上，方便你录制教程、演示视频或讲解类内容。

## 功能

- 摄像头悬浮窗始终置顶
- 简单轻量
- 适合快速录屏和讲解场景
- 界面极简

## 系统要求

- macOS 14.0 及以上

## 安装方式

1. 下载 `floatcamera.dmg`
2. 打开 DMG 文件
3. 将 **FloatCamera** 拖入 **Applications（应用程序）**
4. 在应用程序中打开它

如果首次打开时 macOS 出现安全提示：
- 右键点击应用
- 选择 **打开**
- 再次确认即可

## 摄像头权限

FloatCamera 需要摄像头权限来显示实时画面。

如果还没有授权：
- 打开 **系统设置**
- 进入 **隐私与安全性**
- 打开 **相机**
- 给 **FloatCamera** 开启权限

这是一个极简版本，用于分享和体验基本功能。
后续可能不会持续更新或长期维护。
